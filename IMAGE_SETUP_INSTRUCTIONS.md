# Background Images Setup Instructions

## Required Images

You need to add the following background images to the `glamora-app/assets/` directory:

### 1. Onboarding Screens (4 images)
- **onboarding-1.jpg** - Beauty services at doorstep theme
- **onboarding-2.jpg** - Professional beauty expert theme
- **onboarding-3.jpg** - Easy booking/calendar theme
- **onboarding-4.jpg** - Happy customers/reviews theme

### 2. Welcome Screen (1 image)
- **welcome-bg.jpg** - Premium beauty/spa atmosphere

### 3. Login Screen (1 image)
- **login-bg.jpg** - Elegant beauty/wellness theme

## Image Specifications

- **Format**: JPG (for better performance)
- **Dimensions**: 1080x1920 pixels (portrait orientation)
- **Quality**: High quality, but optimized for mobile (aim for 200-500KB per image)
- **Style**: 
  - Professional beauty/spa photography
  - Soft, elegant lighting
  - Vibrant but not oversaturated colors
  - Premium, luxurious feel
  - Avoid busy backgrounds (gradients will overlay)

## Recommended Image Sources

1. **Unsplash** (https://unsplash.com/)
   - Search terms: "beauty salon", "spa", "makeup artist", "nail salon", "hair salon"
   - Free for commercial use

2. **Pexels** (https://www.pexels.com/)
   - Search terms: "beauty treatment", "cosmetics", "wellness", "luxury spa"
   - Free for commercial use

3. **Pixabay** (https://pixabay.com/)
   - Search terms: "beauty", "glamour", "cosmetics", "salon"
   - Free for commercial use

## Quick Setup (Temporary Placeholders)

If you want to test the UI immediately, you can use solid color gradients as temporary placeholders.
The app will work with any images, but for the best experience, use high-quality beauty-themed photos.

## After Adding Images

1. Place all images in `glamora-app/assets/`
2. Restart the Expo development server
3. The images will be automatically loaded by the app

## Color Palette Suggestions

For best results, choose images that complement the app's color scheme:
- **Primary**: #FFD700 (Gold/Yellow)
- **Secondary**: Soft pinks, purples, or neutrals
- **Accent**: Black, white, or deep burgundy

Images with these tones will create a cohesive, premium beauty brand experience.
